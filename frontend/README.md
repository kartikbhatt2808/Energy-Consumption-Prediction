# Energy Prediction System - Frontend

## Quick Start

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm start
```

3. Build for production:
```bash
npm run build
```

## Features
- Hybrid AI Model (Prophet + XGBoost + Rule-based)
- State-wise energy predictions
- Appliance-level consumption breakdown
- SHAP explainability
- Interactive visualizations

## Tech Stack
- React 18
- Recharts for visualizations
- Lucide React for icons
